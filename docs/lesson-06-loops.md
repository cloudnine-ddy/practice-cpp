# Lesson 06 - Loops

## What I learned

- Some Syntax of C++:

    - for loop syntax:
      ```
      for (int i = 1; i <= n; i++)
      {
        do something with i;
      }
      ``` 

    - while loop syntax:
      ```
      int i = 0;                    # declare the `i` before the loop

      while (i <= n)
      {
        do something with i;
        i++;                        # update the `i` inside the loop
      }
      ```
