# Lesson 04 - Operators and Expressions

## What I learned

- Some Syntax of C++:

    - `+` -> addition operator  
    
    - `-` -> subtraction operator  
    
    - `*` -> multiplication operator  
    
    - `/` -> division operator  
        - if both numbers are integers,it ONLY keep the integer part of the result, without rouding
        
        - if any of the numbers is a floating-point number, it will be normal division (floating-point division)
    
    - `%` -> modulus operator (remainder of division)  
    
    - `=` -> assignment operator
