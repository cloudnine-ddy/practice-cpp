# Lesson 07 - Functions

## What I learned

- Some Syntax of C++:

    - Syntax of function declaration:  
      ```
      return_type function_name(<datatype parameter>, <>, <>, ... [list])
      {
          // function body
      }
      ```  

      like:  

      ```
      int add(int x, int y)
      {
          return x + y
      }
      ```  
    
    - `void` for not returning
