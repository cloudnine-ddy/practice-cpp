# Lesson 12 - Pointer

## What I learned

- Some Syntax of C++:

    - `&variable_name;`  
      get the address of the variable

    - `int* pointerToValue = &value;`  
      declare a pointer variable and assign the address of the value to it
      Notes:
      - `int*` is the data type of the pointer variable
      - it can be `char*`, `float*`, `double*`

    - `*pointerToValue`  
      - is used to dereference the pointer variable
      - meaning to get the value that the pointer is pointing to, not the address

    - `int* emptyPointer = nullptr;`  
      pointer can be null, but references cannot be null

    - `pointerName -> attributeName` 
      is used to access the attribute of the struct, like `structName.attributeName`
